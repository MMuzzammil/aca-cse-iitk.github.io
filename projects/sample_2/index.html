Robot Motion Planning

In this project, we are trying to move a turtlebot from starting point to the destination, in such a way that it tries to avoid all the obstacles in the way.

This is a console application. The user enters the destination cell coordinates on the console which are given as input. This input is passed to a planner that plans the path considering the fixed obstacles there. This path is passed on to the executor that follows the path. Now if the object sees an obstacle in its path, it sends the signal to the replanner which replans the path and passes on to the executor.
Regarding the algorithm, we are using the dstarlite algorithm to plan the path from source to destination and replan it again.Currently, this project is being run on a simulator. We haven’t yet tested it in real time.

A small video of the test will be shown in a while. So, dont worry, be happy.
